# iulms-requester

Make .env file in the directory with following data

> MOODLE_SESSION =
> MOODLE_SESSION_TEST =
> SMTP_AUTH_USER =
> SMTP_AUTH_PASS =
> SMTP_AUTH_HOST =

### Get MOODLE_SESSION Data

go to iulms.edu.pk, login to your account, see website cookies.

setup any smtp server, shared hosting will work too.
